How to use the embed snippet

Paste the generated HTML into any static page or CMS widget. The snippet already loads Leaflet from a CDN and recreates all markers and shapes. 

<!-- Example output -->
<div id="map-123" style="width:100%;height:420px;border:1px solid #e2e8f0;border-radius:12px;"></div>
<link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" crossorigin="" />
<script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js" crossorigin=""></script>
<script>
  const mapConfig = { ...exported data... };
  // map initialisation goes here
</script>
